0712笔记 一，深入对浮动的性质了解 1，浮动脱标，没有块和行内之分： 脱离标准流的三种方法： 1）浮动脱标：float：left； 2）绝对定位：position：absolute； 3）固定定位：position：fixed; eg：span标签是一个标准的行内标签，一旦脱标，就可设置宽高 div是一个标准的块级标签，不设置宽高，默认父亲的宽度,但是一旦脱标，不设置宽度，将自动收缩至内容宽度 2，浮动元素依次贴边 元素都是左浮动 父盒子左边框内部->儿子1->儿子2->…… 如果父盒子剩余的位置已经补足语容纳自己，那么将会依次寻找……3,2,1,父亲的边去贴 3，竖直方向上的margin塌陷现象消失 margin塌陷是标准流的事，浮动脱标了就没这个事儿了 4，让出标准流 注意：意义不大，工作中覆盖使用的是定位以及层级，不会使用这个小技巧 5,文字环绕效果 浮动元素在标准流元素上边，标准流元素中的文字会环绕浮动元素 二，浮动消除 1，父级元素不能被浮动的子元素撑出高度 自由都在标准流中才可以背撑出高度 通过某种方法，让父亲认识浮动的儿子，就可以撑出高度 2，治病的偏方 有一个属性叫;overflow:hidden; 可以解决浮动儿子不能为父亲撑出高度的问题 overflow：hidden；本来的意义； overflow 溢出的意思，hidden影藏的意思 溢出的部分被隐藏 3，有高度的盒子可以管住自己浮动的儿子 浮动元素是会影响别人的，给父盒子设置高度，可以让浮动的儿子不影响别人，也可以不让别人影响自己的儿子 4，浮动清除的方法1：clear：both； 为什么清除浮动，浮动的元素会影响别人，本来是该换行的，但是由于浮动，变成一行 clear；both；清除的是浮动之间的影响 clear 有三个属性值; clear:left；清除左浮动 clear;right;清除右浮动 clear：both；清除左右浮动 缺点; 1）还是没有撑出高度 2）margin失效 5，清除浮动的方法2：隔墙法 隔墙法非常实用，在网页中两个大部分比如：header和content之间我们使用一堵墙隔开，是的两个之间互不影响，但是margin依然是失效的，我们可以使用这个墙来当margin margin虽然失效，但是可以实用小技巧弥补，但是很重要的一点是，没有高度 6，清除浮动的方法3；内墙法 cl：clear:both; 现在margin好使了，高度也有了，但是还有一点问题，html标签有点多，并且没有语义，看起来不爽 7，清除浮动的方法4：overflow：hidden； 总结：清除浮动的方法有很多，但是很多都只存在于面试中，在工作中； 1）我们总是喜欢在浮动元素的父类上添加overflow：hidden 2）我们喜欢在两个大块之间使用外墙法隔开
