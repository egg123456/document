五，废弃标签介绍
	html把标签负责语义和结构，不管样式，2008年以前，html也是负责样式的，有一些标签不是负责语义的，而是负责样式的，08年以后 
	决定废弃这个负责样式的标签，使用css替代
	<b></b>  加粗
	<u></u>	 下划线
	<i></i>	 斜体
	<del></del>  删除线
	虽然被废弃，但是还是有一定用处，配合css使用
	当前在html中负责语义和结构的标签，常用的一些标签有：
		容器：div,ul(li),ol,dl,table
		放文字：h，p,span,a

七，基本选择器
	选择器就是选择一个元素的方法
	1，标签选择器
		标签选择器是最简单的选择器，就是选择页面上所有相同的标签，使用标签名称进行选择
		注意：选择的是页面上所用相同的标签，而不是其中的某一个
		不管元素层次多深，都能找到
		但是标签选择器实用性不高，因为选择的是所有相同标签，适合设置默认样式
		所有的样式在chrome中控制台的elements右边都是有显示的
	2，id选择器
		针对某一个元素，进行特别的样式设置，需要给定这个元素一个特殊的别名，然后通过这个别名选择
		使用id属性为元素添加唯一标志，在css样式表中利用id选择该元素
		<p id="pp">我是段落</p>
		选择：(必须使用#选择)
		#pp{
			color:red;
		}
		任何元素都可以设置id属性，合法的id属性：英文字母开头（大小写敏感，A和a是两个不同的id），可以有数字，下划线_，短横-
		在同一个页面中，id绝对不能相同，，就算是不同的元素，id也是不能相同，在一个页面上，必须保证id唯一，在js中是必须会出错的
		我们可以综合利用标签选择器和id选择器一起做效果，也就是说同一个标签可以被多个选择器选择，这就是层叠样式表的第一层含义
	3，类选择器
		与id选择器不一样的是，类选择器可以重复
		id选择器是唯一，如果页面上有多个元素想设置相同的样式，id选择器就显得很无力，这是使用类选择器，就是给某一类元素设置相同的类名，通过这个类名给给这一类元素添加样式
		设置类：
			<p class="pp1"></p>
		选择：（使用.选择）
			.pp1{
				color:red;
			} 

		类的命名：英文字母开头（区分大小写），数字，下划线，短横
		注意：类名和元素是多对多的关系，就是说一个元素可以拥有多个类名，一个类名可以被多个元素使用,一个元素使用多个类名，类名之间使用空格隔开
		原子类：一些简单的属性组成的一个类，然后元素自行选择是否使用这个类
		定义类名的时候尽量有语义，比如说：warning，inportant，ad，main
		在页面中使用id还是class？在css中尽可能使用class，在js获取标签尽量使用id
		class注重样式，id注重行为
		css样式中尽可能多的使用原子选择器，一个元素尽可能多的继承类名，每一个类名尽可能少的写内容
