三，定位
	css最重要的三个技术：浮动 盒模型 定位三种负责页面的布局
	其他的属性都是杂碎，背景 文字 颜色····
	1，相对定位
		position：relative；
		相对于自己原来的位置，进行位置移动，移动的时候需要位置信息
		top：距离上边多远 向下移动
		left：距离左边多远 向右移动
		bottom：距离下边多远 向上移动
		right：距离右边多远 向左移动
		这四个属性可以任意组合，但是每个组合只能从top和bottom中选一个，从left和right中选一个
		top：100px；
		left;100px;
		==
		bottom:-100px;
		right:-100px;
		相对定位特点：老家留坑，形影分离
		相对定位用处并不大：
			1）微调元素
			2）子绝父相
		相对定位不投标
	2，绝对定位(四角定位)
		position:absolute;
		绝对定位的元素，脱离标准文档流，使用top bottom left right定位，参考点是页面的左上角
		body默认有8px的边框，移动时切忌加上这8px
		1）定位参考点
			相对定位参考点：原来位置的左上角
			绝对定位参考点：如果使用top定位，参考点是页面的左上角或者右上角（不是浏览器窗口的左上角）如果使用bottom进行定位，就是浏览器首屏左下角页面中的位置
			绝对定位的元素，元素整个飘起来了，脱离了标准流
		2，祖先元素当做参考点
			子绝父相：子元素绝对定位，父元素相对定位
			一个盒子可以以祖先元素作为参考点，距离自己最近的已经定位的祖先元素（可以是相对绝对固定定位）
			绝对定位无视父类的padding
		3，绝对定位元素的居中
			绝对定位脱离标准流，就不能使用margin：0 auto；居中
			width:100px;
			position:absolute;
			left:50%;
			top:0;
			margin-left:-50px;
		4)应用
	3，固定定位
		position：fixed；
		固定定位参考点就是浏览器的角
		固定定位的参考点一定是浏览器，我们无法让一个元素作为固定定位参考点，这个参考点是不会跑的
		3种脱标方法：
			浮动，绝对定位，固定定位
			这三种脱标方法，只能有一种在同一个元素身上，不存在一个元素即浮动又定位在浮动中，父类给了overflow：hidden；父类就可以被浮动的儿子撑出高度
			但是在定位中，绝对定位和固定定位的儿子永远不能给父类撑出高度

			如果相对定位和浮动同时存在，无伤大雅
			如果绝对定位和浮动在一起，会议绝对定位为准

四，z-index属性	层级属性
	用于设置压盖顺序，谁压着谁
	默认的压盖顺序；
		1）定位元素压盖无定位元素
		2）如果都定位，那么HTML元素写在后边的压盖前边的

	z-index没有单位的，在都定位了的元素中，数字大的压盖数字小的，只有定位的元素才能使用z-index
	
	父级同样大小的，子级无论大小与否都同父级一样被覆盖。
