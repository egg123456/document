二，继承性和层叠性
	1，继承性
		css规定，有一些样式，给某一个元素设置了，他的后代元素同样会拥有这个样式属性，这就叫做继承性	
		那些样式可以继承？
			color   字体颜色
			text-系列的   文字的修饰 text-align:center;
			font-系列：字体系列  font-size:15px;
			line-系列：文本行系列   line-height：20px;
			一般情况与文本相关的可以继承
		border，background-color等都是不能继承的
		‘层叠’样式表，cascading的第二层含义就是继承性
	2，层叠性
		什么是层叠性？
			就是处理冲突的能力，也就是当多个选择器同时选中一个元素，听谁的？
		同一个元素：
			
我是段落

			
			此时选择的是id选择器，这里是因为css有一套严密的权重法律
		选择器的基本权重：id的权重>class的权重>标签的权重
		id    100
		class 010
		标签  001
		如果样式是继承来的，权重是0
		如果权重一样听谁的？
			1）如果都选中了这个元素，以style列表中后出现的选择器为准
			2）如果都是通过继承得来的，就近原则，看到达目标元素的路径
		权重相同时
		1，都选中元素时
		继承style样式中写在最下面的
		2，没有选中元素时
		如果都是继承得来的，就近原则，看到大目标元素的路径
	！important 权重无限大(但还可以再加)
	总结：权重问题计算顺序
		1）选中情况（也即是路径相同）：
			（1）看有没有！important（针对单个样式）
			（2）计算权重
			（3）执行style最下面的

		2）没有选中情况（即是继承来的权重为零，路径不相同）：
			（1）就近原则
			（2）路径一样时;计算权重
			（3）权重再一样时：执行style最下面的
	作业;!important有效期
		答：只有路径相同时，！important才有效，且只是对它标注的style中的哪一行样式有效，其他的按选择器权重从新选择
