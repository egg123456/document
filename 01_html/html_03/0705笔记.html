	容器级标签有哪些？在HTML4.01基础上，容器及标签有div，li,dl,dd,dt,td,其他的标签基本都是文本级标签

二，图片
	1，基本使用
		语法：<img src='……' alt='图片显示错误' title='图片'>
		img：image 图片
		src：source 资源
		alt：提示
		title：标题
		img标签是一个自封闭标签，单标签<img />
		img可插入的图片格式：jpg，jpeg,png,gif,bmp,不能插入psd格式
	2，图片的宽高
		<img src='' width='100' height='200'>
		属性width：宽度
		height：高度

		如果只给其中一个值，另一个值会按比例自动变化

	3，图片img标签是文本级标签，可以认为是‘文本’
		文本及标签一般会并排排列，除了p标签，并排的过程中元素与元素之间会有默认的间隙，如何消除间隙？
		<img src=''>
		<img src=''>
		<img src=''>
		消除间隔：
		<img src=''><img src=''><img src=''>
	4,绝对路径和相对路径
		1）绝对路径
			所有一http://开头的或者以/开头的路径都是绝对路径
			eg：<img src='https://www.baidu.com/img/bd_logo1.png'>
			上边路径是可以使用
			<img src='/Users/maindek/Desktop/最新课件/最新课件/01 HTML/03 html课件(iqd)/html新版/HTML第2天/案例笔记/images/shanghai.jpg'>
			这种路径不建议使用，可移植性太低
		2）相对路径
			以当前HTML文件为基础，前往寻找图片的位置的这个路径叫做相对路径，相对的是当前的HTML文件
			eg:<img src='images/longlong.jpg'>
			<img src='images/aa/bb/1.jpg'>
			<img src='../../HTML第2天/images/1.png'>
			../只能出现在开头
			优点：可移植性很高，将整个文件夹移植到任何地方，这个相对路径是不会变得
			推荐使用

三，超链接a标签
	1，基本使用
		语法：<a href='temp.html'>我是超链接</a>
		站内链接，相对路径
		<a href='http://www.sina.com.cn' title=''>……</a>
		绝对路径
		<a href='temp.html' target='_blank'>target</a>
		在新的窗口打开链接
		<a href='temp.html' target='_blank'><img src='images/longlong.jpg' width='200'></a>
	2，页内锚点
		语法：<a name='maodian' href='temp.html'>temp</a>
			跳转到这个锚点:<a href='#maodian'><a>
			设置锚点时：在a标签中可以使用name和id设置，不是a标签必须使用id设置
	3，不同页面锚点跳转
		跳转方法：<a href='temp.html#maodian'>跳转到temp页面中的maodian位置</a>
		在temp.html中设置锚点：
			<a name='maodian' id='maodian'>a标签的锚点</a>
			<h1 id='maodian'>h1锚点</h1>

			</li>

五，表单
	表单中的所有元素都要放在form标签中
	语法：<form>
		//表单元素
	</form>
	form就是英语表格的意思。这个标签是一个功能型的标签，不是一个结构型的标签。form标签是后台哥哥来进行配置的。Ajax课程的第一天，我们讲解PHP语言，将介绍更多的表单提交的原理。那时候你就知道了form表单的两个属性method、action，现在先不说。
	1，单行输入框
		<input type='text' value='main'>
	2,密码输入
		<input type='password'>
	3，单选框
		<input type='radio' id='sex'>

		<label for='sex'>男</label>
		<label>
			<input type='radio'>女
		</label>
	4,复选框

六，表格
	1,基本表格
		table：表格，桌子
		tr：table row 行
		td：table dock 小格
		每一个tr就是一行，每一个td就是一个小方格
		<table border="1">
			<tr>
				<td>hello</td>
				<td>hello</td>
				<td>word</td>
			</tr>
		</table>
		border:边框
		<caption>名单</caption>写在表格内部的第一行，表格的标题
		th:table header 表头  自动文本居中，加粗
	2，细线表格
		<style>
			table{
				/*合并边框*/
				border-collapse: collapse;
			}
		</style>
		cellspacing：边框之间的距离
		cellpadding：边框与文字之间的距离
	3，合并单元格
		rowspan:行跨度合并   竖着合并
		colspan:列跨度合并   横着合并
		这两个属性都是td的属性
		<table>
			<caption></caption>
			<thead>
				<tr>
					<th></th>
				</tr>
			</thead>
			<tbody>
				<tr>
					<td></td>
				</tr>
			</tbody>
		</table>
		在td，th中可以有div，span，注释，转义字符等内容
		br，hr这些在css中再说
		

		表格border只不为零时，会有2px的单元格间距，可用cellspacing属性设置为零或用style中border-collapse：collapse；消除
