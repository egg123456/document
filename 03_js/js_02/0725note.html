二，变量类型的转换
	1，string-->number
		弹出输入框
			prompt(
			'请输入一个数字：')；

		语法：var a = prompt("提示文本",'默认值');
		这个方法的返回值是字符串

		parseInt();将字符串解析成整形，如果内容是小数，不会四舍五入直接把小数部分扔掉，
		如果这个字符串不是数值，就会取开头的数字，开头不是数字则为nan，此方法也可用于进制解析

		parseFloat():将字符串解析成浮点型
			console.log(parseInt('-12.99px'));//-12
			console.log(parseInt('px-12.99px'));//nan

			console.log(parseInt('15e6',10));//e不是数字，取头，十进制15等于15
			console.log(parseInt('0xf',16));
			console.log(parseInt('f',16));
			console.log(parseInt('f',8));//8进制里没有8以上的，所以为nan

		Number();也可以转换，但Number里面必须是规范的数值，否则为nan

		String();将数字转换为字符串

五，逻辑运算
	逻辑运算符有三个
		&&	与 
		||	或
		！	非
	正统的是布尔与布尔之间的运算，结果也是布尔值
	
运算顺序：非 与 或

	console.log(false || true  && !true || false);

	&&特点；
		var a = 1;
		var b = 2;
		var c = ++a && ++b;
		console.log(a,b,c);//2,3,3    由于++a不等于0，所以执行++b并把值赋给c

	短路语法：

		1）如果是一个与运算，比如a&&b，a如果是假，那么就不会管b，结果就是FALSE或者直接输出a
		2）如果是一个与运算，比如a&&b，a是TRUE，直接把b当做结果输出
		3）如果是一个或运算，比如a||b，a是FALSE，直接把b输出
			若a为TRUE，那就不管b，直接把a当做结果输出

	
负性：FALSE null 0 nan 字符串空‘’ undefined

	正性：除了上边的都是正性的
