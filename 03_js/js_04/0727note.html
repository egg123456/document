一，循环语句
	在js中流程控制语句有两种：分支语句，循环语句，依靠这两种语句能完成所有程序
	循环语句是一系列反复执行直到复合特定条件的命令
	循环语句能够解决大量类似语句书写的问题
	1，for循环
		语法：
			for( 定义循环变量及初值;设定循环条件;循环变量变化式 )
			for( var i=0; i<num ; i++ ){
				sentens;
			}
		流程：
			1）声明循环变量赋值
			2）检测循环变量，是否满足条件
			3）如果满足条件，执行循环代码，不满足，跳出循环
			4）满足条件循环代码执行完毕，执行循环变量自增
			5）重复2,3,4
		var i=1;
		for( alert('hello'); i < 12; i+=3,alert(i),alert('world')){
			console.log(i);//1 4 7 10
		}
	2，穷举思想

二，for循环深入
	1，for循环中的break和continue语句(减少for循环的执行次数)
		break语句：跳出循环--只能跳出当前for循环
			如果想要让一个break终止外层循环，使用label语法
			waiceng:for (var i = 0; i <= 10; i++) {
						for (var j = 0; j <= 10; j++){
							console.log(i,j);
							if (j==5) {
								break waiceng;//关闭内外循环 00 01 02 03 04 05结束
							}
						}
					}
		continue语句：for循环会终止本次循环后边的语句，直接跳到下一次循环
			只能终止内部循环，想要终止外循环，加label语法
	2，do while循环--先做在判断
			do{
				
			}while(条件)
		如果while()为空，报错unexpected token
	3，white循环--和for差不多
		while(false){
			console.log('lalal');
		}
