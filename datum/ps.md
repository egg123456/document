# 将一张图片以任意大小放在别一张图片上
+ 打开ps
+ 将底层图片拖入ps
+ 将上层图片拖入ps
+ 拉图片边缘放大缩小
+ ctrl+鼠标拖动 到指定位置
+ 合并图层（ctrl+e 合并所有图层；ctrl+shift+e 合并可见图层）
+ 保存为你想要的格式
